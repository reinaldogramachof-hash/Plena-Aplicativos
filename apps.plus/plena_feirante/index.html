Instalação do Sistema

Torne o App Disponível OFFLINE

Instale no celular para usar sem internet, igual um app normal:

Android (Chrome)
  1. Abra esta página no Chrome
  2. Toque nos 3 pontinhos ☰
  3. Toque em "Instalar app"
Windows (Chrome)
  1. Clique no ícone (+) na barra
  2. Clique em "Instalar"
2

Configuração Inicial (5 Minutos)

PASSO 1: Dados do Negócio

  1. Clique em "Configurações" (ícone da engrenagem)
  2. Preencha os dados: Nome da Feira, CPF/CNPJ e Telefone
  3. Defina o endereço: Para aparecer nos recibos

PASSO 2: Cadastro de Produtos

  1. Vá em "Produtos" no menu lateral
  2. Clique em "Novo Produto"
  3. Defina: Nome, Categoria, Preço de Venda e Estoque Inicial

Dica de Ouro: Categorias

Organize seus produtos por categorias (Frutas, Legumes, Verduras) para facilitar a busca na hora da venda e ter relatórios mais precisos.

Frutas Legumes Verduras
3

Realizando Vendas

Venda Rápida (PDV)

  1. 1. Selecione os produtos clicando neles
  2. 2. Ajuste as quantidades se necessário
  3. 3. Clique em "Finalizar Venda"
  4. 4. Escolha a forma de pagamento (Dinheiro, PIX, Cartão)

Venda com Cliente

  1. 1. Selecione o cliente antes de finalizar
  2. 2. O sistema gera histórico de compras
  3. 3. Facilita o controle de fiados (se habilitado)
  4. 4. Fidelize seus clientes com descontos
